What is C++

C++ is a high level, general purpose, object oriented programming language which is used for developing desktop applications, video games and other complex applications. It's a lot to take in at once, so let's break it down step by step.

"C++ is a high level language" but what does it mean?

It basically means that the syntax(the methodology to write statements) are in a language which can be understood by people.
"C++ is a general purpose programming language"

general purpose programming language implies that the language can be used to develop applications for variety of tasks.
"C++ is an object oriented"

Object oriented languages are languages which support the use of classes and objects. Further information on the definition and the applications of classes and objects is provided in chapter.
